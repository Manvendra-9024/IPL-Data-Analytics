# IPL Data Analyst Project

## Overview
Comprehensive analysis of Indian Premier League (IPL) cricket data covering team performance, player stats, match outcomes, and season trends.

## Tools & Technologies
- **Python / Pandas** – Data processing and statistical analysis
- **SQL** – Match and player database queries
- **Power BI** – Interactive IPL dashboards
- **Tableau** – Season trend visualizations
- **Excel** – Data source and summary tables

## Analysis Modules
- Team win/loss analysis by season and venue
- Top batsmen and bowlers by season
- Toss impact analysis
- Head-to-head matchup statistics
- Player of the Match trends
